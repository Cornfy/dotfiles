# dotfiles
- 该项目用于备份我的 Arch Linux 的常用配置文件。
- 主要用于 Wayland 环境。

#### .profile 文件
- 该文件为环境变量，可将该文件放在 `$HOME` 目录下。

#### .config 文件夹
- 此文件夹对应 `$HOME/.config` 文件夹，内含各软件的配置文件。
  - alacritty：     一个终端模拟器
  - fcitx5：        常用的输入法框架
  - fish：          另一个 shell，自带历史命令预测等功能
  - mako：          一个用来发送通知的工具，常用于窗口管理器（WM）环境
  - neofetch：      一个用于查看系统信息的工具
  - nvim：          Neovim，一个 Vim 的 GUI 版本
  - ranger：        一个终端下的文件管理器
  - rofi：          一个程序启动器，常用于 WM 环境
  - swaylock：      一个锁屏工具，常用于 WM 环境
  - waybar：        一个状态栏工具，常用于 WM 环境
  - wlogout:        一个注销、重启工具，常用于 WM 环境

#### .local 文件夹
- 此文件夹对应 `$HOME/.local` 文件夹，内含我自定义的 `.desktop` 文件。
